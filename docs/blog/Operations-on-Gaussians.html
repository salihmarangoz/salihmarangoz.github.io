Univariate Gaussian PDF¶

$$ \mathcal N(x; \mu, \sigma^2) = \frac{1}{\sigma\sqrt{2\pi}} \exp \bigg(-\frac{1}{2} \frac{(x-\mu)^2}{\sigma^2} \bigg) $$

In [2]:
def gaussian_pdf(x, mu=0, sigma=1):
    denom = (1/(sigma*np.sqrt(2*np.pi)))
    return  denom * np.exp(-0.5*((x-mu)**2)/(sigma**2))

mu = 3
sigma = 2
x = np.linspace(-10, 10, 1000)
y = gaussian_pdf(x, mu, sigma)
_=plt.plot(x, y)

$$ \varphi(x) = \frac{e^{-\frac{x^2}2}}{\sqrt{2\pi}} $$

Z-Transform¶

There are two ways;

$x, \mu, \sigma^2 \rightarrow \text{gaussian pdf} \rightarrow$ y

or,

$x, \mu, \sigma^2 \rightarrow \text{z transform} \rightarrow \text{normal pdf} \rightarrow y$

$$ \mathcal N(x,\mu,\sigma^2) = \varphi(z) $$

In [5]:
def z_transform(x, mu, sigma):
    z = (x-mu) / sigma
    return z

# Example(1):
x = 1.0
mu = 2.0
sigma = 1.5

# source: https://malishoaib.wordpress.com/2014/04/02/python-code-and-normal-distribution-writing-cdf-from-scratch/
# modified or python numpy
def erf(x): # John D. Cook's implementation.http://www.johndcook.com
    # constants
    a1 =  0.254829592
    a2 = -0.284496736
    a3 =  1.421413741
    a4 = -1.453152027
    a5 =  1.061405429
    p  =  0.3275911
 
    # Save the sign of x
    sign = x>0
    x = np.abs(x)
 
    # Formula 7.1.26 given in Abramowitz and Stegun.
    t = 1.0/(1.0 + p*x)
    y = 1.0 - (((((a5*t + a4)*t) + a3)*t + a2)*t + a1)*t*np.exp(-x*x)
 
    return (np.array(sign, dtype='int')*2-1) *y

def phi(x): # aka cdf
    return 0.5*( 1.0 + erf(x/np.sqrt(2)) )

Gaussian Affine Transform¶

$$ z \sim \mathcal N(x, a\mu + b, a^2 \sigma^2) $$

In [9]:
def gaussian_affine_transform(mu, sigma, a, b):
    mu_ = a*mu + b
    sigma_ = a * sigma   # Caution! It is not a**2 here, because sigma is std not variance.
    return mu_, sigma_

sigma = 2

Gaussian Product¶

The inverse of variance is called information.

$$ i^2 = \frac{1}{\sigma^2} $$

Gaussian product:

$$ \mathcal N(x;\mu_1, \sigma^2_1) . \mathcal N(x;\mu_2, \sigma^2_2) = \mathcal N(x;\mu_3, \sigma^2_3) $$

$$ \mu_3 = \frac{\mu_1 \sigma^2_2 + \mu_2 \sigma^2_1}{\sigma^2_1 + \sigma^2_2}=\frac{\mu_1 i^2_1 + \mu_2 i^2_2}{i^2_1 + i^2_2} $$

$$ \sigma_3^2 = \frac{\sigma^2_1 \sigma^2_2}{\sigma^2_1 + \sigma^2_2} = \frac1{i_1^2 + i_2^2} $$

In [10]:
def gaussian_product(mu, sigma):
    info2 = 1 / (sigma**2)
    mu_ = np.mean(mu * info2)
    sigma_ = np.sqrt( 1/(np.sum(info2)) )
    return mu_, sigma_

mu1 = 0
sigma1 = 1
mu2 = 2
sigma2 = 1.5

mu = np.array([mu1, mu2])
sigma = np.array([sigma1, sigma2])
new_mu, new_sigma = gaussian_product(mu, sigma)
print("New mu:", new_mu)
print("New sigma:", new_sigma)


# Plot
x = np.linspace(-7, 7, 100)

y2 = gaussian_pdf(x, mu2, sigma2)

Gaussian Convolution¶

$$ \mathcal N(x;\mu_1, \sigma^2_1) * \mathcal N(x;\mu_2, \sigma^2_2) = \mathcal N(x;\mu_3, \sigma^2_3) \\ $$

$$ \mu_3 = \mu_1 + \mu_2 \\ $$

$$ \sigma_3^2 = \sigma_1^2 + \sigma_2^2 $$

Multivariate Gaussian PDF¶

For $\vec x$, $\vec \mu \in R^m$ and non-singular $\Sigma \in R^{m \times m}$

$$ \mathcal N(\vec x; \vec \mu, \Sigma) = \frac{1}{\sqrt{(2\pi)^m \det \Sigma}} \exp^{-\frac{1}{2} (\vec x - \vec \mu)^T\Sigma^{-1}(\vec x - \vec \mu)} $$

In [13]:
def gaussian_m_pdf(x, mu, Sigma, inv_method=np.linalg.pinv):
    m = len(mu)
    denom = np.sqrt((2*np.pi)**m * np.linalg.det(Sigma))
    expr = np.exp(-0.5 * (x-mu).T @ inv_method(Sigma) @ (x-mu))
    return expr / denom


x = np.array([0, 1])
mu = np.array([0, 1])
Sigma = np.array([[1,0.5],[0.5,3]])

Gaussian Affine Transform¶

Suppose

$$ \vec x \sim P_{Gauss}(\vec \mu, \Sigma) $$

and let

$$ \vec z = A \vec x + \vec b $$

then

$$ \vec z \sim P_{Gauss}(A\vec\mu + \vec b, A \Sigma A^T) $$

new_mu = A @ mu + b

Gaussian Product¶

$$ \mathcal N(\vec x;\vec \mu_1, \Sigma_1) . \mathcal N(\vec x;\vec \mu_2, \Sigma_2) = \mathcal N(\vec x;\vec \mu_3, \Sigma_3) \\ $$

$$ \Sigma_3 = (\Sigma_1^{-1} + \Sigma_2^{-1})^{-1} \\ $$

$$ \mu_3 = \Sigma_3 (\Sigma_1^{-1} \vec \mu_1 + \Sigma_2^{-1} \vec \mu_2) $$

In [16]:
def gaussian_m_product(mu1, Sigma1, mu2, Sigma2):
    inf1 = np.linalg.inv(Sigma1)
    inf2 = np.linalg.inv(Sigma2)
    Sigma3 = np.linalg.inv(inf1 + inf2)
    mu3 = Sigma3 @ ( inf1 @ mu1 + inf2 @ mu2 )
    return mu3, Sigma3

mu1 = np.array([-1, 1])
Sigma1 = np.array([[0.1,0],[0,3]])
mu2 = np.array([2, 2])
Sigma2 = np.array([[3,0.2],[0.2,0.1]])

mu3, Sigma3 = gaussian_m_product(mu1, Sigma1, mu2, Sigma2)
print("New mu:", mu3)
print("New Sigma:\n", Sigma3)
